13.4.1 Гнучкість елемента:

$$\lambda= \frac{l_{ef} }{i} = \frac{ {{l_ef}} }{ {{geometry.i}} } \times100= {{lambda}}$$

$$\overline{ \lambda } = \lambda \sqrt{ \frac{R_y}{E} } = {{lambda}} \sqrt{ \frac{ {{R_y}} }{ {{E}} } } = {{lambdaUndeground}} $$
